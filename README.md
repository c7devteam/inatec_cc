# ActiveMerchant Inatec Gateway

Active Merchant inatec payment gateway. 

## Installation

Add this line to your application's Gemfile:

    gem 'active_merchant_inatec'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install active_merchant_inatec
